🧮 Fatoração: Escrevendo Como Produto

🎯 O que é fatoração?

Fatoração é o processo de transformar uma soma ou diferença em um produto. É o inverso dos produtos notáveis.

Exemplo:
Sabemos que (x + 3)(x – 2) = x² + x – 6
Agora, queremos fazer o contrário:
x² + x – 6 = (x + 3)(x – 2)

Isso é fatorar!



✅ Caso 1: Fator Comum em Evidência

Quando todos os termos têm um fator em comum, colocamos ele "em evidência".

ax + ay = a(x + y)

Exemplo numérico: 15 + 10 = 5×3 + 5×2 = 5(3 + 2)

Exemplo algébrico: 6x + 9 = 3·2x + 3·3 = 3(2x + 3)

Com letras: 4x² + 8x = 4x(x + 2)



✅ Caso 2: Agrupamento

Usado quando há quatro termos. Agrupamos dois a dois e colocamos em evidência.

Passo a passo:
• Separe em dois grupos
• Coloque o fator comum de cada grupo em evidência
• Se o parêntese for igual, coloque-o em evidência

Exemplo: ax + ay + bx + by
= a(x + y) + b(x + y)
= (x + y)(a + b)



✅ Caso 3: Trinômio Quadrado Perfeito

É o inverso do quadrado da soma ou diferença.

a² + 2ab + b² = (a + b)²
a² – 2ab + b² = (a – b)²

Como identificar:
• Primeiro e último termo são quadrados perfeitos
• Termo do meio é o "2 vezes o primeiro pelo segundo"

Exemplo: x² + 6x + 9
→ x² é quadrado de x
→ 9 é quadrado de 3
→ 6x = 2·x·3 → sim!
Portanto: x² + 6x + 9 = (x + 3)²



✅ Caso 4: Diferença de Quadrados

É o inverso do produto da soma pela diferença.

a² – b² = (a + b)(a – b)

Como identificar: dois termos ao quadrado separados por sinal de menos.

Exemplo: x² – 16 = (x + 4)(x – 4)
25 – y² = (5 + y)(5 – y)



💡 Aplicações Práticas

A fatoração é usada para:

  • Simplificar frações algébricas: cancelar termos iguais
  • Resolver equações: quando o produto é zero, um dos fatores é zero
  • Calcular raízes de funções quadráticas
  • Facilitar cálculos em álgebra e cálculo


🧠 Dica Final: Como Reconhecer o Caso

  1. Tem fator comum? → Use evidência
  2. Tem 4 termos? → Tente agrupamento
  3. Tem 3 termos e começa com quadrado? → Verifique se é trinômio quadrado perfeito
  4. Tem dois quadrados com sinal de menos? → Diferença de quadrados
  5. Pratique bastante — reconhecimento vem com a prática!


🧠 Conclusão

A fatoração é uma ferramenta essencial na matemática. Dominá-la permite simplificar expressões, resolver equações e entender melhor a álgebra.

Lembre-se: fatorar é escrever como produto. E você agora sabe como fazer isso!
