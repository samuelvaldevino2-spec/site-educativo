Gabarito

  1. Isótopos
  2. Isótonos
  3. Isóbaros
  4. Prótons + Nêutrons
  5. Cátion

  11. Porque possuem o mesmo número atômico (mesmo número de prótons), mas diferentes números de massa (diferentes quantidades de nêutrons).
  12. Isóbaros são átomos de elementos diferentes com o mesmo número de massa, como ⁴⁰Ar e ⁴⁰K. Isótonos são átomos com o mesmo número de nêutrons, como ¹⁴C (8 nêutrons) e ¹⁵N (8 nêutrons).
  13. São isótonos, porque isótonos são definidos pelo mesmo número de nêutrons, mesmo que os elementos e massas sejam diferentes.
  14. É importante para entender propriedades radioativas, datação (como carbono-14), medicina nuclear, e para prever comportamento químico e físico de elementos.
  15. O professor concluiu que são isótopos porque isótopos são átomos do mesmo elemento (mesmo número de prótons) com diferentes números de massa (diferentes quantidades de nêutrons).
